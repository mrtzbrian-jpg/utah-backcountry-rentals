# Gear photos

Drop a photo in this folder for each item and it appears automatically in the app
(home feed, gear detail, and the Build-Your-Pack library). Until a file exists,
the item shows its outdoorsy icon tile instead — so the app never looks broken.

## File names (this is the important part)

Save each photo with the **exact** name below. Any of these formats works —
`.jpg`, `.png`, or `.webp` (the app tries them in that order):

| File name (any of .jpg/.png/.webp) | Product to photograph |
|---|---|
| `osprey-rook-65`        | Osprey Rook 65 backpack |
| `naturehike-cloudup-1`  | Naturehike Cloud Up 1-person tent |
| `kelty-cosmic-20`       | Kelty Cosmic Synthetic 20° sleeping bag |
| `klymit-static-v`       | Klymit Static V Base sleeping pad |
| `brs-3000t`             | BRS-3000T titanium stove |
| `garmin-inreach`        | Garmin inReach Mini 2 |
| `bearvault-bv450`       | BearVault BV450 food canister |
| `anker-10k`             | Anker 10,000mAh power bank |
| `bear-spray`            | Frontiersman Bear Spray |

Example: `images/osprey-rook-65.jpg`

## Photo tips for the best look

- **Square-ish** images work best (the tiles are roughly square).
- Shoot on a **plain white or light-gray background**, or use a **transparent PNG**
  cut-out — the gear sits on a soft light tile, so a clean background blends in.
- The image is centered and fully visible (never cropped), with a little padding.
- ~1000×1000px is plenty; keep files under ~300 KB so the app loads fast.

That's it — add the files and reload. No code changes needed.
